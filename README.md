## What is Figma?

Figma is a **browser-based collaborative design tool** for creating UI/UX designs,
prototypes, and design systems. It supports real-time multiplayer editing — multiple
users can work in the same file simultaneously, like Google Docs for design.

**Founded:** 2012 (by Dylan Field & Evan Wallace)
**Public Launch:** 2016
**Platform:** Browser + Desktop app (Windows, macOS, Linux, Chrome OS)
**License:** Freemium (Free tier available)

---

## Core Features

### 1. Real-Time Collaboration
- Multiple editors in the same file simultaneously
- Comments with @mentions and direct replies
- Version history with named saves
- Shared cursor visibility

### 2. Auto Layout
- Flexbox-like responsive frames
- Automatic spacing and resizing
- Horizontal, vertical, and wrap modes
- Nested Auto Layout for complex responsive components

### 3. Components & Variants
- Master components with propagating overrides
- Variant properties (e.g., Type, State, Size)
- Component libraries shared across files
- Design tokens for color, typography, spacing

### 4. Prototyping
- Native click/hover/drag interactions
- Smart Animate for smooth transitions
- Interactive components (dropdowns, toggles)
- Shareable prototype links (no app required)

### 5. Developer Handoff
- Inspect tab with CSS properties
- Export assets (PNG, SVG, PDF, WEBP)
- Code view for CSS, Swift, Android
- Design token export via plugins

### 6. Figma AI (2026)
- Text-to-layout generation
- Automated layer renaming
- Background removal
- Copy edit suggestions
- Multi-language translation
