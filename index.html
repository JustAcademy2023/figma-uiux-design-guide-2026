What is Figma?

Figma is a browser-based collaborative design tool for UI/UX design, prototyping, and design systems with real-time team collaboration.

Core Features

Real-Time Collaboration

Multiple users edit simultaneously with comments and live cursors.

Auto Layout

Responsive design system similar to Flexbox.

Components

Reusable design systems with variants and tokens.

Prototyping

Interactive UI flows with Smart Animate.

Dev Handoff

Inspect CSS, export assets, and share design tokens.

Figma AI

Auto layout generation, text editing, and smart suggestions.
